## ðŸ“– Tentang Proyek

Tujuan dari proyek ini adalah membangun *landing page* yang sesuai dengan desain yang telah ditentukan (mockup), dengan fokus pada:
- Tata letak yang responsif (Mobile-first / Desktop-first).
- Interaktivitas elemen menggunakan JavaScript (Tab, Accordion, Menu Mobile).
- Validasi formulir sederhana.

Proyek ini sangat cocok untuk melatih kemampuan layouting CSS dan manipulasi DOM dengan JavaScript.

---

## âœ¨ Fitur Utama

Halaman ini dilengkapi dengan fitur-fitur berikut:

1.  **Navigasi Responsif:** Menu hamburger yang berfungsi dengan baik pada tampilan seluler.
2.  **Tab Fitur (Feature Tabs):** Konten yang berubah dinamis saat pengguna memilih tab "Simple Bookmarking", "Speedy Searching", atau "Easy Sharing".
3.  **Accordion (FAQ):** Bagian tanya jawab yang dapat dibuka-tutup (expand/collapse).
4.  **Validasi Email:** Pengecekan input pada kolom "Contact Us" untuk memastikan format email valid sebelum dikirim.
5.  **Efek Hover:** Interaksi visual saat kursor diarahkan ke tombol atau tautan.
